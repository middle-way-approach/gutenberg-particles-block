A Gutenberg block to render a particle effect.

== Description ==

You can use the block inside the Gutenberg editor as you would use a any other block. You can find it in the common section.

<sub>Icon made by [freepik](https://www.flaticon.com/de/autoren/freepik) from www.flaticon.com<sub>

== Installation ==

After you install and activate the plugin you can use it as a block in the Gutenberg editor. You find it under the widgets section.
